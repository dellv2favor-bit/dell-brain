# smart-escalation.js

**Built by Dell on 2026-03-25**

## Why
Trusted contacts (Brandon) hit walls and ask Dell to 'call Rondell' or escalate — Dell can't actually bridge that gap efficiently. Brandon says 'the server keeps crashing' and 'I've tried everything' with no resolution path. (3+ messages from Brandon in this window alone)

## What it does
An escalation formatter that detects when a trusted contact is stuck (keywords: 'won't connect', 'keeps crashing', 'tried everything', 'call Rondell') and automatically sends Rondell a structured summary: who needs help, what they've tried, what's blocking them — so Rondell can respond in one message instead of discovering the problem through a chain.

friction_point | Impact: high | Complexity: small
